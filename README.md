# Bryce Breedlove — Portfolio (Static Site)

## Run locally

### Option A: open directly
- Double-click `index.html`.

### Option B: run a local static server (recommended)
If you have PowerShell:

```powershell
cd path\to\bryce-portfolio
python -m http.server 5173
```

Then open:
- http://localhost:5173

## Files
- `index.html` — content
- `styles.css` — styling (no external build tools)
- `script.js` — small interactions (theme toggle, scroll reveal)
- `Bryce_Breedlove_Resume.pdf` — resume copy used for the site
